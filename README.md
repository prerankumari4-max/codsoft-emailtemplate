# 📧 Responsive Email Template (CodSoft Task)

A modern, responsive email template built using HTML and CSS (table-based layout) to ensure compatibility across major email clients like Gmail, Outlook, and Yahoo Mail.

---

## 🚀 Project Overview

This project is part of the CodSoft internship tasks. The goal was to design a professional email template that is:

* Visually appealing 🎨
* Responsive across devices 📱
* Compatible with multiple email clients 💻

---

## ✨ Features

* 🎨 **Modern Gradient Header** for visual appeal
* 📱 **Responsive Design** using media queries
* 💼 **Professional Layout** (Header, Content, CTA, Footer)
* ⚡ **Call-to-Action Button** for engagement
* 🧠 **Table-based Structure** for email compatibility
* 🔒 Works across Gmail, Outlook, and other clients

---

## 🛠️ Tech Stack

* HTML5
* CSS3 (Inline + Internal Styling)
